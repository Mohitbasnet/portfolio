What is Full Stack Development?

Full stack development refers to the practice of working across all layers of a web application — from what the user sees to how data is stored and served.

Overview

A full stack developer is someone who can build both the frontend (client-side) and backend (server-side) of a web application. The term "stack" refers to the collection of technologies used together to create a complete product.


The three core layers are:

  • Frontend — what users see and interact with (HTML, CSS, JavaScript)
  • Backend — the server logic that processes requests (Node.js, Python, PHP)
  • Database — where data is stored and retrieved (MySQL, MongoDB)

Popular full stack combinations include the MEAN stack (MongoDB, Express, Angular, Node.js) and the LAMP stack (Linux, Apache, MySQL, PHP).

HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It defines the structure and content of a page using a system of elements and tags.

Key Concepts

HTML uses tags (e.g. <h1>, <p>, <div>) to mark up content. Tags usually come in pairs — an opening tag and a closing tag. HTML5 introduced semantic elements like <header>, <nav>, <main>, <section>, and <footer> that give meaning to page structure, improving accessibility and SEO.


  • Semantic HTML — Tags that describe the purpose of content (e.g. <article>, <aside>)
  • Forms — Collect user input using <input>, <textarea>, <select>
  • Tables — Display tabular data using <table>, <tr>, <td>
  • Links & Media — <a> for links, <img> for images, <video> for video

CSS

CSS (Cascading Style Sheets) controls the visual presentation of HTML elements — including layout, colour, typography, spacing, and animations.

Key Concepts

CSS works by selecting HTML elements and applying style rules to them. Styles cascade — meaning more specific rules override more general ones. CSS supports powerful layout systems like Flexbox and CSS Grid, and allows responsive design through media queries.


  • Selectors — Target elements by tag, class (.classname), or id (#idname)
  • Box Model — Every element has margin, border, padding, and content
  • Flexbox — One-dimensional layout (rows or columns)
  • Grid — Two-dimensional layout system
  • Transitions & Animations — Smooth visual effects

Bootstrap Framework

Bootstrap is a popular open-source CSS framework that provides pre-built, responsive components — making it faster to build professional-looking websites without writing CSS from scratch.

Key Concepts

Bootstrap uses a 12-column grid system to create flexible, responsive layouts. It provides ready-made components like buttons, cards, modals, navbars, alerts, and forms that look great on any device. Bootstrap also uses utility classes like mt-3 (margin top), text-center, d-flex to style elements directly in HTML.


  • Grid — col-md-6 spans half the page on medium+ screens
  • Components — Pre-built buttons, cards, navbars, modals, alerts
  • Utilities — Classes for spacing, colour, text, display
  • Responsive — Mobile-first design built in with breakpoints (sm, md, lg, xl)

JavaScript

JavaScript is a programming language that adds interactivity and dynamic behaviour to web pages. It runs directly in the browser and can manipulate HTML, respond to events, and communicate with servers.

Key Concepts

JavaScript is the only language that runs natively in browsers. It uses the DOM (Document Object Model) to access and modify HTML elements in real time. JavaScript supports variables, functions, loops, conditionals, and events to build interactive experiences.


  • Variables — Store data using let, const, or var
  • Functions — Reusable blocks of code
  • DOM Manipulation — document.getElementById(), innerHTML, style
  • Events — React to user actions (onclick, onkeyup, onsubmit)
  • Arrays & Objects — Store and organise multiple values
